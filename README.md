# توضیحات ماشین حساب

## بخش یکم template

من برای ساخت ماشین حساب داخل فایل 
HelloWorld.vue
تغییراتی ایجاد کردم

در قسمت
template
من از تگ
table
در خط 75
برای ساخت یک جدول استفاده کردم و به آن جدول سطر و ستون دادیم
tr
و
td

و در خط 79 یک ورودی
input
و در ادامه دکمه های ماشین حساب را تعریف کردم با تگ
button
و جدول را در خط 110 بستم
